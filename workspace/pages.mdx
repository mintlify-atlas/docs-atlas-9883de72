# Pages

Pages is LobeHub's collaborative writing feature that transforms how you create content with AI. Instead of switching between multiple conversations, Pages brings your AI agents into a single shared document where they can collaborate, iterate, and refine your writing in real-time.

## Why Use Pages?

Traditional AI chat interfaces require you to copy-paste content between conversations, losing context and momentum. Pages changes this by providing:

- **Unified workspace**: All agents work on the same document with shared context
- **Real-time collaboration**: Multiple agents can contribute simultaneously
- **Version control**: Track changes and revert to previous versions
- **Rich formatting**: Full markdown support with syntax highlighting, tables, and more

## Creating a New Page

<Steps>

### Navigate to Pages

Click the "Pages" option in the left sidebar or navigation menu to access your Pages workspace.

### Start a New Page

Click the "New Page" or "+" button to create a blank page. You'll see the page editor interface with:

- A title field at the top
- An emoji picker (optional) to personalize your page
- The main editor canvas below

### Add Your Content

Start typing in the editor. The interface supports:

- **Markdown formatting**: Use `#` for headings, `**bold**`, `*italic*`, etc.
- **Code blocks**: Use triple backticks for code with syntax highlighting
- **Lists and tables**: Format content with bullets, numbers, and tables
- **Links and images**: Embed references and visual content

</Steps>

## Working with Agents in Pages

### The Page Agent Copilot

Pages includes a built-in AI copilot that appears on the right side of the editor. This specialized page agent helps you:

- **Generate content**: Ask it to write sections, paragraphs, or entire documents
- **Refine writing**: Request edits, rewrites, or style improvements
- **Research and expand**: Add facts, examples, or deeper explanations
- **Format and structure**: Organize content with proper headings and flow

### Collaborative Editing Workflow

The page agent understands the full context of your document and can:

1. **Read the entire page**: It sees all content, not just recent messages
2. **Make targeted edits**: Modify specific sections while preserving the rest
3. **Track changes**: Show you what's being added or modified
4. **Maintain consistency**: Keep voice, tone, and style aligned throughout

### Using Multiple Agents

You can involve different specialized agents in your page:

- Switch agents in the copilot panel to get different perspectives
- Ask a technical writer to improve clarity
- Bring in a subject matter expert for accuracy
- Use a creative writer for engaging narratives

## Page Management

### Organizing Your Pages

Access all your pages from the Pages list view:

- **Search**: Filter pages by title or content
- **Sort**: Order by creation date or last modified
- **Pin favorites**: Keep important pages at the top

### Page Actions

Each page supports these operations:

- **Rename**: Click the title to change the page name
- **Change emoji**: Update the page icon for quick identification
- **Duplicate**: Create a copy to start variations or templates
- **Delete**: Remove pages you no longer need
- **Share**: Export or share page content (if available in your workspace)

### Auto-Save

Pages automatically saves your work:

- Changes are saved every few seconds while you type
- A status indicator shows "Saving..." or "Saved" in the header
- You can safely close the page without losing progress

## Best Practices

### Start with Structure

Begin by outlining your document with headings:

```markdown
# Main Title

## Introduction

## Key Points

### Point 1

### Point 2

## Conclusion
```

Then ask the agent to fill in sections progressively.

### Use Clear Instructions

When working with the page agent, be specific:

- ❌ "Make this better"
- ✅ "Rewrite this section to be more concise and add a concrete example"

### Iterate in Sections

Rather than generating an entire document at once:

1. Have the agent write one section
2. Review and refine it
3. Move to the next section with context from what's already written

This produces higher quality results than bulk generation.

### Leverage Templates

Create template pages for common document types:

- Meeting notes
- Project plans
- Research summaries
- Blog posts

Duplicate templates to start new documents quickly.

## Keyboard Shortcuts

Speed up your workflow with keyboard shortcuts:

- `Cmd/Ctrl + S`: Manually trigger save
- `Cmd/Ctrl + B`: Bold selected text
- `Cmd/Ctrl + I`: Italic selected text
- `Cmd/Ctrl + K`: Insert link

### Documentation

Write technical documentation and guides:

- Structure complex information clearly
- Add code examples and explanations
- Maintain consistent terminology

### Research Notes

Compile and synthesize research:

- Organize findings from multiple sources
- Summarize key points
- Create structured overviews

### Planning Documents

Develop project plans and strategies:

- Outline goals and milestones

## Integration with Knowledge Bases

Pages can connect to your knowledge bases:

- Link pages to specific knowledge base collections
- The page agent can reference and cite sources from the knowledge base
- Keep related documents and research organized together

## Tips for Effective Collaboration

1. **Set context early**: Tell the agent about your audience, purpose, and desired outcome
2. **Review incrementally**: Check each section before moving forward
3. **Use the chat history**: Reference previous conversations to maintain continuity
4. **Combine human and AI**: Write the parts you know best, let AI handle research and expansion
5. **Experiment freely**: Pages are your workspace - try different approaches

## Limitations to Keep in Mind

- **Token limits**: Very long documents may exceed agent context windows
- **Formatting consistency**: Always review AI-generated formatting
- **Fact-checking**: Verify claims and statistics independently
- **Voice preservation**: Your unique voice may need reinforcement in edits
