# Quickstart guide

This guide will walk you through getting started with LobeHub, from creating your account to having your first conversation with an AI agent.

<Note>
You'll be up and running in less than 5 minutes. No technical knowledge required.
</Note>

## Choose your deployment method

You have three options to access LobeHub:

<CardGroup cols={3}>
  <Card title="Cloud (recommended)" icon="cloud">
    Use the hosted version at lobehub.com - no setup required
  </Card>
  <Card title="Self-hosted" icon="server">
    Deploy on Vercel, Docker, or other platforms for full control
  </Card>
  <Card title="Desktop app" icon="desktop">
    Download the native app for peak performance without browser limitations
  </Card>
</CardGroup>

<Tip>
For this quickstart, we'll use the cloud version. You can migrate to self-hosted later without losing your data.
</Tip>

## Step 1: Create your account

<Steps>
  <Step title="Visit LobeHub">
    Navigate to [lobehub.com](https://lobehub.com) in your web browser
  </Step>
  <Step title="Sign up">
    Click "Sign Up" and choose your authentication method:
    - Email and password
    - OAuth (GitHub, Google, Discord)
    - Magic link (passwordless)
    - Passkey (biometric authentication)
  </Step>
  <Step title="Verify your account">
    Complete the verification process based on your chosen authentication method
  </Step>
</Steps>

## Step 2: Configure your first AI model

Before you can start chatting, you need to configure at least one AI model provider.

<Steps>
  <Step title="Open settings">
    Click your avatar in the top right corner and select "Settings"
  </Step>
  <Step title="Navigate to providers">
    Go to **Settings → Language Models → Model Providers**
  </Step>
  <Step title="Add your API key">
    Choose a provider (OpenAI, Anthropic, Google, etc.) and add your API key:

    <Accordion title="Where to get API keys">
      - **OpenAI**: Visit [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
      - **Anthropic**: Visit [console.anthropic.com](https://console.anthropic.com)
      - **Google AI**: Visit [makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)
      - **Local models**: Install [Ollama](https://ollama.ai) for free local LLMs
    </Accordion>
  </Step>
  <Step title="Select a model">
    Choose a default model from your configured provider (e.g., GPT-4, Claude 3.5 Sonnet)
  </Step>
</Steps>

<Note>
LobeHub supports over 40 AI model providers. You can configure multiple providers and switch between them anytime.
</Note>

## Step 3: Start your first conversation

Now you're ready to chat with AI!

<Steps>
  <Step title="Return to chat">
    Click the chat icon in the left sidebar or press `⌘+1` (Mac) / `Ctrl+1` (Windows)
  </Step>
  <Step title="Send your first message">
    Type a message in the input box at the bottom and press Enter. Try something like:
    
    ```
    Hello! Can you explain what LobeHub is and how I can use it effectively?
    ```
  </Step>
  <Step title="Review the response">
    Watch as the AI responds in real-time. You can:
    - Copy the response
    - Regenerate if you want a different answer
    - Edit your message and retry
    - Continue the conversation naturally
  </Step>
</Steps>

<Tip>
You can switch models mid-conversation by clicking the model name at the top of the chat window.
</Tip>

## Step 4: Create your first agent

Agents are specialized AI assistants configured for specific tasks. Let's create one!

<Steps>
  <Step title="Open the agent builder">
    Click the **+ New Agent** button in the left sidebar
  </Step>
  <Step title="Describe your agent">
    Simply describe what you want your agent to do. For example:
    
    ```
    Create a writing assistant that helps me write clear, 
    concise technical documentation. It should focus on 
    clarity, use active voice, and follow best practices 
    for developer documentation.
    ```
    
    The Agent Builder will automatically configure the agent based on your description.
  </Step>
  <Step title="Customize (optional)">
    You can further customize:
    - **Name and avatar**: Give your agent a unique identity
    - **System prompt**: Refine the instructions
    - **Model selection**: Choose the best model for this task
    - **Skills**: Add plugins and MCP integrations
    - **Knowledge base**: Upload relevant documents
  </Step>
  <Step title="Start using your agent">
    Click "Create" and start chatting with your new specialized agent
  </Step>
</Steps>

## Step 5: Explore key features

Now that you've had your first conversation, explore these powerful features:

### Upload files and create knowledge bases

<Steps>
  <Step>
    Click the paperclip icon in the chat input to upload files
  </Step>
  <Step>
    Drag and drop documents (PDF, DOCX, TXT), images, audio, or video files
  </Step>
  <Step>
    Ask questions about your uploaded content - the AI can read and analyze it
  </Step>
</Steps>

### Use voice conversations

<Steps>
  <Step>
    Click the microphone icon in the chat input
  </Step>
  <Step>
    Speak your message - it will be automatically transcribed (Speech-to-Text)
  </Step>
  <Step>
    Enable TTS in settings to hear responses read aloud (Text-to-Speech)
  </Step>
</Steps>

### Try branching conversations

<Steps>
  <Step>
    Hover over any message in your conversation
  </Step>
  <Step>
    Click the branch icon to create an alternative conversation path
  </Step>
  <Step>
    Choose between Continuation Mode (keeps context) or Standalone Mode (fresh start)
  </Step>
</Steps>

### Add plugins and skills

<Steps>
  <Step>
    Go to **Settings → Skills & Plugins**
  </Step>
  <Step>
    Browse the marketplace with 40+ plugins and thousands of MCP integrations
  </Step>
  <Step>
    Install skills for web search, code execution, API integrations, and more
  </Step>
  <Step>
    Enable plugins in your agent's settings to extend its capabilities
  </Step>
</Steps>

## Common workflows

<AccordionGroup>
  <Accordion title="Research and analysis">
    1. Create a research agent with web search enabled
    2. Upload relevant documents to the knowledge base
    3. Ask the agent to analyze and synthesize information
    4. Export findings as PDF or Markdown
  </Accordion>
  
  <Accordion title="Content creation">
    1. Create a writing agent with your preferred style
    2. Use branching to explore different writing approaches
    3. Leverage artifacts to preview generated content
    4. Iterate with the agent until you're satisfied
  </Accordion>
  
  <Accordion title="Code assistance">
    1. Create a programming agent with code execution skills
    2. Upload your project files or codebase context
    3. Use Chain of Thought to see the reasoning process
    4. Copy code snippets with syntax highlighting
  </Accordion>
  
  <Accordion title="Team collaboration">
    1. Create an agent group for complex projects
    2. Assign different agents to different roles
    3. Use Pages to collaborate with multiple agents on documents
    4. Schedule automated tasks to run while you're away
  </Accordion>
</AccordionGroup>
