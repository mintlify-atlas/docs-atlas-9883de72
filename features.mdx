## Agent creation and management

### Agent Builder

Create sophisticated AI agents in seconds with our AI-powered Agent Builder. Simply describe what you need, and LobeHub automatically configures the perfect agent setup.

**Key features:**
- **Auto-configuration** - Intelligent system prompts, model selection, and capability setup
- **505+ marketplace agents** - Pre-built agents for every use case
- **Custom agents** - Full control over behavior, tools, and knowledge
- **Agent forking** - Build on existing agents from the community

<Note>
  See [Agent Builder](/agents/agent-builder) for detailed configuration options.
</Note>

### Agent Groups

Take collaboration to the next level with Agent Groups - multiple agents working together on complex tasks.

**Orchestration modes:**
- **Sequential** - Agents work one after another
- **Parallel** - Multiple agents work simultaneously
- **Iterative** - Agents refine each other's work
- **Debate** - Agents discuss and reach consensus

**Learn more:** [Agent Groups](/concepts/agent-groups)

## Intelligence and memory

### Personal Memory

LobeHub features white-box memory that builds a deep understanding of you:

- **Continual learning** - Agents remember your preferences, style, and context
- **Transparent memory** - View and edit what agents remember
- **Structured storage** - Memory organized by type (preferences, patterns, history)
- **Privacy-first** - Full control over what's remembered

**Learn more:** [Personal Memory](/concepts/memory)

### Chain of Thought

Watch AI reasoning unfold step-by-step with our Chain of Thought visualization:

- **Transparent reasoning** - See how agents reach conclusions
- **Step-by-step breakdown** - Understand complex problem-solving
- **Educational value** - Learn from AI's thought process
- **Debugging aid** - Identify where reasoning goes wrong

**Learn more:** [Chain of Thought](/features/chain-of-thought)

## Content and collaboration

### Pages

Collaborative writing workspace powered by AI:

- **Page Agent Copilot** - AI assistant that helps you write
- **Rich editing** - Full markdown support with live preview
- **Multi-agent writing** - Multiple agents contributing to documents
- **Knowledge integration** - Pull from knowledge bases automatically

**Learn more:** [Pages](/workspace/pages)

### Knowledge Bases

Transform your documents into searchable, AI-accessible knowledge:

- **Semantic search** - Find information by meaning, not just keywords
- **Multiple formats** - PDFs, Word docs, text files, code, images
- **Automatic chunking** - Intelligent document processing
- **RAG integration** - Agents use knowledge bases automatically

**Learn more:** [Knowledge Bases](/agents/knowledge-base)

### Artifacts

Create and iterate on content in real-time:

- **Code generation** - React components, HTML, SVG graphics
- **Diagrams** - Mermaid flowcharts and visualizations
- **Documents** - Formatted content with live preview
- **Iterative refinement** - Edit and regenerate seamlessly

**Learn more:** [Artifacts](/features/artifacts)

### Database Options

Choose your data storage:

- **Local database** - PGLite for single-user deployments

- **CRDT sync** - Multi-device synchronization (experimental)
- **Full control** - Your data stays with you
