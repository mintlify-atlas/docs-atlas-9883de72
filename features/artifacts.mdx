LobeHub integrates Claude Artifacts capabilities, enabling AI to create real-time, interactive content that goes beyond text responses. Generate visualizations, interactive applications, and documents that appear in a dedicated preview pane.

## What are Artifacts?

Artifacts are substantial, self-contained pieces of content created by AI that deserve their own dedicated space. Instead of inline code blocks, artifacts open in a separate panel where you can:

- View live previews of generated content
- Interact with applications and visualizations
- Edit and iterate on the output
- Export or copy the final result

When the AI creates an artifact, it automatically appears in the artifacts panel on the right side of your screen.

## Using Artifacts

### Creating Artifacts

Artifacts are created automatically when you ask the AI to generate substantial, self-contained content.

**Trigger artifact creation:**

```
"Create a React calculator app"
"Generate an SVG chart showing quarterly sales"
"Build an interactive timeline of World War II"
"Design a landing page for a coffee shop"
```

**What qualifies as an artifact:**
- Substantial content (more than a few lines)
- Self-contained and functional
- Something you might want to iterate on
- Content that benefits from a dedicated preview

**What doesn't need artifacts:**
- Short code snippets or examples
- Quick explanations or answers
- Simple text responses

### Viewing Artifacts

When an artifact is created:

1. The artifacts panel automatically opens on the right
2. Content renders in real-time as it's generated
3. A preview appears showing the live result
4. You can switch between preview and code view

**Display modes:**

<Tabs>
  <Tab title="Preview Mode">
    Shows the rendered output:
    - Interactive applications run live
    - SVG graphics display at full quality
    - HTML pages render completely
    - Diagrams appear formatted

    **When to use:**
    - Testing functionality
    - Viewing final output
    - Presenting to others
    - Interacting with the content
  </Tab>

  <Tab title="Code Mode">
    Shows the source code:
    - Syntax highlighted
    - Formatted and readable
    - Easy to copy
    - Includes comments and structure

    **When to use:**
    - Understanding implementation
    - Copying code to your project
    - Making manual edits
    - Learning from examples
  </Tab>
</Tabs>

### Iterating on Artifacts

Continue the conversation to refine artifacts:

```
"Add a dark mode toggle"
"Make the chart responsive for mobile"
"Change the color scheme to blue and green"
"Add error handling to the form"
```

Each iteration updates the artifact in real-time, letting you see changes immediately.

### Exporting Artifacts

Depending on the artifact type, you can:

- **Copy code** - Copy the complete source to clipboard
- **Download SVG** - Save graphics as .svg files
- **Download PNG** - Export graphics as images
- **Copy as image** - Copy rendered output to clipboard
- **Save HTML** - Download complete web pages

## Tips for Working with Artifacts

<AccordionGroup>
  <Accordion title="Be specific in requests">
    Detailed prompts produce better artifacts. Specify colors, layout, functionality, and features upfront.
  </Accordion>

  <Accordion title="Iterate incrementally">
    Make one change at a time to maintain stability. Large changes can introduce bugs or inconsistencies.
  </Accordion>

  <Accordion title="Test interactivity">
    For interactive artifacts, test all features before exporting. Ask for bug fixes or edge case handling.
  </Accordion>

  <Accordion title="Use preview and code modes">
    Switch between modes to understand both the visual result and implementation details.
  </Accordion>

  <Accordion title="Save your favorites">
    Copy or download artifacts you want to keep. They remain in your conversation history but are easier to access when saved separately.
  </Accordion>
</AccordionGroup>

<Warning>
  Artifacts run in a sandboxed environment for security. Some features requiring external API calls or sensitive permissions may not work.
</Warning>

<Note>
  Complex artifacts may take a few moments to generate. Watch the real-time rendering as the AI creates your content.
</Note>
