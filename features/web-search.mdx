LobeHub's web search integration allows AI agents to access current information from the internet, providing up-to-date answers beyond their training data cutoff.

## Overview

With web search enabled, AI agents can:

- **Access real-time information** - Get current news, data, and facts
- **Verify information** - Check latest developments and updates
- **Cite sources** - Provide links to original information
- **Research topics** - Gather information from multiple sources
- **Stay current** - Answer questions about recent events

## How It Works

When you ask a question that requires current information:

1. **Query generation** - AI formulates optimized search queries
2. **Web search** - Searches are performed across the internet
3. **Source gathering** - Relevant sources are retrieved and analyzed
4. **Information synthesis** - AI reads and synthesizes findings
5. **Cited response** - Answer includes source citations

All of this happens automatically when the AI determines web search would be helpful.

## Using Web Search

### Automatic Search

Web search activates automatically for questions like:

```
"What's the current price of Bitcoin?"
"Who won the World Cup in 2024?"
"What are the latest developments in AI?"
"What's the weather forecast for New York?"
```

The AI recognizes these require current data and searches automatically.

### Explicit Search Requests

You can also explicitly request web search:

```
"Search the web for recent reviews of the iPhone 15"
"Look up the latest research on climate change"
"Find current mortgage rates in California"
"Search for tutorials on Next.js 14"
```

### Search Refinement

Refine searches with follow-up questions:

```
Initial: "What are the top programming languages in 2024?"
Follow-up: "Search for job market trends for Python developers"
Follow-up: "Find salary data for senior Python engineers"
```

Each search builds on the conversation context.

## Search Quality

### Optimized Queries

The AI automatically optimizes search queries:

- **Reformulation** - Converts questions into effective search terms
- **Multiple angles** - Generates several queries for comprehensive coverage
- **Specificity** - Balances broad and specific search terms
- **Recency** - Includes time-relevant terms when needed

### Source Selection

Not all search results are used:

- **Relevance filtering** - Only pertinent sources analyzed
- **Quality assessment** - Prioritizes authoritative sources
- **Diversity** - Includes multiple perspectives
- **Recency weighting** - Favors current information when appropriate

### Information Synthesis

The AI processes search results intelligently:

- **Cross-referencing** - Verifies info across sources
- **Conflict resolution** - Handles contradicting information
- **Context integration** - Combines search results with existing knowledge
- **Citation tracking** - Links claims to specific sources

## Understanding Citations

When web search is used, responses include citations:

### Citation Format

Citations typically show:
- **Source title** - Page or article name
- **Domain** - Website hosting the information
- **URL** - Direct link to source
- **Relevance snippet** - Preview of relevant content

## Privacy & Search

<Warning>
  Web searches may be logged by search providers. Avoid searching for sensitive or private information.
</Warning>

**Privacy considerations:**

- Search queries sent to search providers
- Source websites may track visits
- Search history may be retained
- Use private/incognito mode if concerned

**Best practices:**

- Don't search for personal information
- Avoid searches containing sensitive data
- Be mindful of search content
- Review privacy policies of search providers

## Limitations

**Search has limitations:**

- **Paywall content** - Cannot access subscription-only content
- **Real-time data** - Slight delay in indexing very recent information  
- **Regional restrictions** - Some content may be geo-restricted
- **Search engine limits** - Constrained by search provider capabilities
- **Processing time** - Web search adds latency to responses

**When search might not help:**

- Purely creative tasks
- Personal advice without factual basis
- Theoretical or hypothetical scenarios
- Topics requiring opinion rather than facts
